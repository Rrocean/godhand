# GodHand CLI Enhanced v4.0 快速开始

## 功能亮点

1. **自由问答模式** - 用自然语言询问如何使用
2. **功能自检** - 自动检测所有功能是否可用
3. **复合指令** - 一次执行多个操作
4. **对话记忆** - 记住你的使用习惯

## 启动方式

```bash
cd C:\Users\admin\clawd\godhand
python main_cli_enhanced.py
```

---

## 使用示例

### 1. 基础命令

```
GodHand> 打开记事本
GodHand> 输入 Hello World
GodHand> 按回车
GodHand> 截图
```

### 2. 复合指令（多个操作一次执行）

```
GodHand> 打开记事本 然后输入Hello 然后按回车
GodHand> 打开计算器 然后输入1+1 然后按等于
GodHand> 打开画图 然后画个圆
```

### 3. 自由问答（询问如何使用）

```
GodHand> 如何打开记事本
GodHand> 怎么截图
GodHand> 你能做什么
GodHand> 如何打开画图并画圆
```

### 4. 系统检查

```
GodHand> check
```
