## 🎯 功能演示

### 1. AI 自主代理

```python
from core import AIAgent

agent = AIAgent(name="My Agent")

# 注册技能
agent.register_skill("open_app", lambda target: {"output": f"打开 {target}"})

# 运行任务
result = agent.run("打开浏览器，搜索 Python 教程")
# AI 自动规划：分析 → 执行 → 验证 → 反思
```

### 2. 语音控制

```python
from core import VoiceController

vc = VoiceController()

# 语音交互模式
vc.interactive_mode()
# 🎤 说出命令，系统自动识别并执行
```

### 3. 云端同步

```python
from core import CloudSync, CollaborationRole

sync = CloudSync()
sync.register_device({"name": "User", "email": "user@example.com"})

# 共享工作流
sync.share_workflow("wf_001", "自动备份", steps=[...])

# 邀请团队成员
sync.invite_team_member("team@example.com", "Team Member", CollaborationRole.EDITOR)
```

### 4. 视觉理解

```python
from core import VisualEngine

engine = VisualEngine()

# 检测UI元素
elements = engine.detect_elements(screenshot)

# 智能定位
target = engine.locate_element("保存按钮", screenshot)
```

## 🚀 快速开始

### 安装

```bash
git clone https://github.com/godhanddev/godhand.git
cd godhand
pip install -r requirements.txt
```

### 启动

```bash
# 启动主程序
python main_v3.py

# 或启动示例
python examples/universe_demo.py
```

### 使用

```bash
# Web 界面
open http://localhost:8000

# API 调用
curl -X POST http://localhost:8000/api/execute \
  -H "Content-Type: application/json" \
  -d '{"command": "打开计算器"}'
```
